## VS Code Commands
* ```OPTION + Z```: Wrap text
* ```COMMAND + S```: save your progress
* ```COMMAND + D```: selects all instances of highlighted text
* ```SHIFT + OPTION + DOWN```: Copy down
* ```SHIFT + OPTION + A```: Comments out all highlighted text
